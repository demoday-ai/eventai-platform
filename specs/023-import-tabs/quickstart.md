# Quickstart: Import Data Tabs

## Verification Checklist

1. Open `/import` page → see 4 numbered tabs: "1. Событие", "2. Проекты", "3. Эксперты", "4. Гости"
2. Tab "Событие" is active by default
3. No event exists → Event tab shows creation form with "Создать" button
4. Fill name + dates → click "Создать" → event created, confirmation shown
5. Switch to "Проекты" tab → upload form visible (existing functionality)
6. Switch to "Эксперты" tab → upload form visible
7. Switch to "Гости" tab → upload form with subtype dropdown visible
8. Delete event → tabs 2-4 show hint "Сначала создайте мероприятие" with link to tab 1
9. Upload projects on tab 2 → switch to tab 3 → switch back to tab 2 → result preserved
10. Edit event on tab 1 → Dashboard updates countdown

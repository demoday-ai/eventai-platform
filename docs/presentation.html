Проблема и ICP

Кто и когда страдает

ICP: Организаторы Demo Day, конференций, хакатонов

  • Расписание вручную за ночь до DD
  • 80 студентов не попали в расписание
  • Нужно 60 экспертов, активных — 30
  • Нет аналитики и follow-up

Боль гостей (партнёры, HR, инвесторы)

  • 330 проектов в 10 залах
  • Видят <20% контента
  • Пропускают 68% релевантного
  • Нет персонализации
Цена боли: организатор тратит 8+ часов на расписание, гость уходит разочарованным, бизнес-партнёр не находит нужный проект → потерянные контакты и сделки
Решение

Почему нельзя просто спросить ChatGPT?

× Нет данных

ChatGPT не знает ваши 330 проектов, залы, расписание

× Нет действий

Не может отправить напоминание, записать на слот, запросить контакт

× Нет стабильности

Каждый запрос — новый контекст, нет памяти профиля гостя

× Нет контроля

Организатор не видит аналитику, не управляет контентом

EventAI = специализированный агент с данными события + действиями + памятью + админкой

Технология

Где AI даёт ценность

Пайплайн рекомендаций

  • TF-IDF — быстрый отбор по тегам
  • LLM re-ranking — Claude/GPT переранжирует топ-20 с учётом контекста профиля
  • Tool-calling — агент вызывает инструменты (поиск, Q&A, сравнение)

Стек

  • python-telegram-bot 21.x
  • FastAPI + PostgreSQL
  • OpenRouter (Claude 3.5, GPT-4o)
  • React 19 + TanStack Query
Ключевое: LLM понимает почему гостю интересен проект (контекст, задачи), а не просто совпадение тегов

Экономика

Кто платит и за что

Модель

  • B2B — организаторы платят за событие
  • От 100 000 ₽ за мероприятие
  • Годовая подписка для регулярных
  • Enterprise — индивидуально

Unit-экономика

  • Cost на гостя: ~5-10 ₽ (LLM API)
  • Cost на событие: ~3-5K ₽
  • Маржа: >90%

Снижение: кэширование, локальные модели, batch-запросы

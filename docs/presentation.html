Технология

Где AI даёт ценность

Пайплайн рекомендаций

  • TF-IDF — быстрый отбор по тегам
  • LLM re-ranking — переранжировка топ-20
  • Tool-calling — агент вызывает инструменты
  • AI-кластеризация — автораспределение по залам

Стек

  • python-telegram-bot 21.x
  • FastAPI + PostgreSQL
  • Celery + RabbitMQ (async)
  • OpenRouter (Claude 3.5, GPT-4o)
  • React 19 + TanStack Query
Ключевое: LLM понимает почему гостю интересен проект, а не просто совпадение тегов. Автокластеризация — ключевая AI-ценность для организаторов.

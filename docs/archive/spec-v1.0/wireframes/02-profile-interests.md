# Wireframe: Профиль интересов

> **Платформа:** Mobile (Telegram bot, 375px)
> **Контекст:** После выбора роли
> **Цель пользователя:** Задать интересы для подбора проектов

---

## Wireframe

```
╭────────────────────────────────────╮
│ Navigator                     ◯ ⚙ │
├────────────────────────────────────┤
│ Шаг 2 из 4                          │
│                                     │
│ Выберите темы и индустрии:          │
│                                     │
│ [✓] AI / ML       [ ] FinTech       │
│ [ ] CV            [✓] EdTech        │
│ [ ] Agents        [ ] HR            │
│ [ ] Industry      [ ] Research      │
│                                     │
│ Стадия проекта:                     │
│ (●) MVP   (○) Pilot   (○) Scale     │
│                                     │
│ Свободные интересы:                 │
│ ╭────────────────────────────────╮  │
│ │ Например: LLM, RAG, healthcare │  │
│ ╰────────────────────────────────╯  │
│                                     │
│ ╭────────────────────────────────╮  │
│ │ Продолжить                     │  │
│ ╰────────────────────────────────╯  │
╰────────────────────────────────────╯
```

---

## Аннотации

| # | Элемент | Поведение |
|---|---------|-----------|
| 1 | Чек‑листы тем | Можно выбрать несколько |
| 2 | Стадия | Один выбор (radio) |
| 3 | Свободные интересы | Уточнение поиска |
| 4 | CTA «Продолжить» | Переход к рекомендациям |

# Wireframe: Подтверждение участия

> **Платформа:** Mobile (Telegram bot, 375px)
> **Контекст:** Эксперт/участник подтверждает участие
> **Цель пользователя:** Указать тему/время и подтвердить участие

---

```

├────────────────────────────────────┤

│                                     │
│ Секция/комната:                     │
│ ╭────────────────────────────────╮  │

---

## Аннотации

| # | Элемент | Поведение |
|---|---------|-----------|
| 1 | Секция/комната | Выбор места выступления |
| 2 | Дата/время | Указывается пользователем |
| 3 | Тема | Краткое описание для орг‑команды |
| 4 | CTA «Подать заявку» | Отправляет заявку со статусом «подано» |
| 5 | Статус | Показывает текущий статус заявки |

---

## Состояния

### Подтверждено

```
╭────────────────────────────────────╮
│ ✅ Заявка подтверждена              │
│ Дата/время: 06.02 14:30            │

```
╭────────────────────────────────────╮
│ ⚠ Укажите дату и время              │
╰────────────────────────────────────╯
```

## Переходы

- Подать заявку → экран «Статус участия»
- Подтверждение → экран «Мой список»

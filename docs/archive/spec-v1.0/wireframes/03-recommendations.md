> **Контекст:** После заполнения интересов

## Аннотации

| # | Элемент | Поведение |
|---|---------|-----------|
| 1 | Карточка проекта | Краткое описание + CTA |
| 2 | «В избранное» | Добавляет в список 3–5 проектов |
| 3 | «Скорректировать интересы» | Возврат к профилю |
| 4 | «Мой список» | Список сохранённых проектов |
| 5 | «Продолжить» | Переход к подтверждению участия (для экспертов/участников) или завершению онбординга |

## Переходы

- В избранное → Экран «Мой список»
- Продолжить → Экран «Подтверждение участия» (эксперт/участник)
- Продолжить → Завершение (гость)

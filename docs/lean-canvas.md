# Lean Canvas: AI-агент-куратор DemoDay

## 1. Проблема

**Топ-3 проблемы:**
1. Гостям Demo Day сложно найти релевантные проекты среди 150+ (6 параллельных залов, 2 дня) за ограниченное время -- гость видит <20% проектов
2. Фоллоу-ап после мероприятия хаотичный -- контакты теряются, договорённости забываются
3. Разные сегменты гостей (инвесторы, HR, партнёры) имеют разные задачи, но получают одинаковый опыт

**Существующие альтернативы:**
- Demo-Hero (@jugru_conf_bot) -- tinder-пары докладов, поверхностное профилирование, нет follow-up
- Бумажная программа / PDF-каталог
- Сарафанное радио на месте
- Ручная рассылка организаторами

## 2. Сегменты пользователей

**Основные:**
- Индустриальные партнёры / заказчики
- Инвесторы
- HR / нанимающие менеджеры

**Вторичные:**
- Менторы / жюри
- Организаторы (core-команда Хаба)

## 3. Уникальное ценностное предложение

AI-агент, который за 2 минуты собирает персональную программу Demo Day под задачи гостя, помогает с Q&A и формирует follow-up пакет с контактами и next steps.

**High-level concept:** "Персональный куратор Demo Day в Telegram"

## 4. Решение

1. **Умный онбординг** -- гость формулирует интерес (отрасль/стек/задача/формат партнёрства), агент профилирует
2. **Персональная программа** -- подборка проектов с брифами, ранжированная под запрос
3. **Q&A помощник** -- вопросы для авторов, матрица сравнения проектов
4. **Организация встреч** -- запись на слоты 1:1 или групповой Q&A
5. **Follow-up пакет** -- конспект, контакты, next steps, шаблоны писем/LOI

## 5. Каналы

- Telegram-бот (основной канал)
- QR-код на мероприятии
- Рассылка от организаторов перед Demo Day
- Веб-версия (опционально)

## 6. Потоки доходов

_Требует валидации:_
- B2B: организаторы платят за платформу (SaaS)
- Freemium: базовая навигация бесплатно, расширенный follow-up за подписку
- Лидген: комиссия за успешные матчи гость-проект

## 7. Структура затрат

- AI API (LLM для агента)
- Серверная инфраструктура
- Разработка и поддержка
- Интеграция с данными о проектах

## 8. Ключевые метрики

- Конверсия: зашёл в бота -> получил программу
- Кол-во проектов в подборке, на которые гость реально пошёл
- Кол-во организованных встреч 1:1
- Follow-up: сколько контактов конвертировались в действие (письмо, звонок)
- NPS гостей
- Повторное использование на следующем мероприятии

## 9. Нечестное преимущество

- Доступ к данным проектов AI Talent Camp (первый клиент = организаторы кэмпа)
- AI-first подход vs. rule-based решения (Demo-Hero)
- Глубокое профилирование под задачи, а не поверхностный tinder

## Методология

- **Reach** — сколько пользователей затронет за один Demo Day
- **Impact** — влияние на ключевую метрику (0.25 = минимум, 0.5 = низкий, 1 = средний, 2 = высокий, 3 = массивный)
- **Confidence** — уверенность в гипотезе (100% = подтверждено данными/цитатой, 80% = сильные сигналы / ТЗ автора, 50% = предположение, 20% = слабый сигнал)
- **Effort** — трудозатраты в person-неделях
- **RICE Score** = (Reach × Impact × Confidence) / Effort

> Все фичи включены в демо по требованию ТЗ ("все 5 ролей, вау-эффект"). Приоритет разработки по RICE — если не хватает времени, режем снизу.

1. H4 — Напоминалки (780)
2. H2 — Подтверждение участия студентов (495)
3. H1 — Автокластеризация проектов (335)
4. H6 — Предобработка и рассылка ОС (264)

**Should have (RICE 50-200) — разрабатываем вторыми:**

6. H16 — Подтипы гостей для аналитики (66) — минимальный effort

**Should have (RICE 50) — разрабатываем вторыми (продолжение):**
8. H10 — Q&A-помощник для гостей/бизнеса (50) — подтверждён Олегом

**Nice to have (RICE < 50, но в ТЗ) — разрабатываем третьими:**
9. H5 — Овервью проектов для экспертов (33)
10. H8 — Персональная программа для гостей (27) — подтверждён Олегом + Настей (5/5)
11. H11 — Follow-up для гостей (27) — подтверждён Настей (5/5), скачок с 13

**Won't have (out of scope):**

## Примечание: CustDev vs ТЗ

| Источник | Кол-во гипотез | Суммарный RICE | Уверенность |
|----------|---------------|----------------|-------------|
| CustDev (интервью с организатором) | 8 | 2093 | 80-100% |
| CustDev + интервью с экспертом | 2 | 88 | 100% |
| CustDev + интервью с Олегом Шатовым | 4 | 117 | 80-100% |
| CustDev + интервью с гостем (Настя) | 3 | 66 | 60-100% |
| Анализ данных | 1 | 66 | 60% |

CustDev-фичи доминируют по RICE (организатор + студенты = массовый reach). Интервью #3 (Олег Шатов) повысило confidence бизнес-фич с 50-80% до 80-100%. Интервью #4 (Анастасия Гапеева) — первая прямая валидация гостевых фич: H8 и H11 подтверждены на 100%, H15 снижена до 60% (гость не видит ценности формальных 1:1). Фичи из ТЗ (гости, бизнес) имеют низкий reach (~15-50), но высокий impact для сегмента и обязательны для демо.

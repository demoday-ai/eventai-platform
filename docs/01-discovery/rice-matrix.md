# RICE-матрица: приоритизация гипотез и фич

_По итогам Customer Discovery интервью #1 (организатор), ТЗ автора идеи (Д. Ботов), анализа экспертного сообщества (294 эксперта) и данных регистрации (190 гостей)._

> **Версия:** 2.0
> **Дата:** 2026-02-01
> **Изменения v2.0:** 5 ролей (+ Бизнес/партнёр), разделение Q&A (гости vs эксперты), новые гипотезы H14-H17, актуализация H7/H8/H10/H11

## Методология

- **Reach** — сколько пользователей затронет за один Demo Day
- **Impact** — влияние на ключевую метрику (0.25 = минимум, 0.5 = низкий, 1 = средний, 2 = высокий, 3 = массивный)
- **Confidence** — уверенность в гипотезе (100% = подтверждено данными/цитатой, 80% = сильные сигналы / ТЗ автора, 50% = предположение, 20% = слабый сигнал)
- **Effort** — трудозатраты в person-неделях
- **RICE Score** = (Reach × Impact × Confidence) / Effort

**Must have (RICE > 200) — разрабатываем первыми:**
1. H4 — Напоминалки (780)
2. H2 — Подтверждение участия студентов (495)
3. H1 — Автокластеризация проектов (335)

**Nice to have (RICE < 50, но в ТЗ) — разрабатываем третьими:**
8. H10 — Q&A-помощник для гостей/бизнеса (40)
9. H5 — Овервью проектов для экспертов (27)

11. H17 — Бизнес-follow-up (18)
12. H15 — Встречи 1:1 (16) — для демо упрощаем до "запросить контакт"
13. H8 — Персональная программа для гостей (13)

**Won't have (out of scope):**

- H13 — Транскрибация записей DD

## Примечание: CustDev vs ТЗ

| Источник | Кол-во гипотез | Суммарный RICE | Уверенность |
|----------|---------------|----------------|-------------|
| CustDev (интервью с организатором) | 8 | 2093 | 80-100% |
| ТЗ автора (Ботов) | 6 | 118 | 50-80% |
| Анализ данных | 1 | 66 | 60% |

CustDev-фичи доминируют по RICE (организатор + студенты = массовый reach). Фичи из ТЗ (гости, бизнес) имеют низкий reach (~15-50), но высокий impact для сегмента и обязательны для демо.

## Риски и допущения

- RICE для H1-H9 основаны на **одном интервью** с организатором. Нужна валидация с другими оргами
- RICE для H10-H17 основаны на **ТЗ автора** — не валидированы с конечными пользователями (гостями, бизнес-партнёрами)
- Reach для бизнес-партнёров (~15) — грубая оценка, зависит от формата DD
- Effort — грубые оценки, уточнить после проектирования архитектуры
- H7 (единая точка входа) имеет RICE 88, но де-факто блокирует все остальные фичи — разрабатывается первой

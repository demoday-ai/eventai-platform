# Пример взаимодействия с Demo-Hero (@jugru_conf_bot)

Реальный лог общения с ботом Demo-Hero на прошлом Demo Day. Показывает текущий UX и его ограничения.

---

## Онбординг

```
Пользователь: /start

Demo-Hero bot:
Привет, Дмитрий Горбунов! Я Demo-Hero -- бот-помощник на защитах проектов Demo Day.

Я помогу тебе составить твой личный "маршрут" на Demo Day, чтобы попасть именно
на те доклады, которые будут максимально интересны, и напомню об их начале,
чтобы ничего не пропустить.

Хочешь попробовать?
```

Далее бот спрашивает **пол** и **"чем занимаешься"** -- это всё профилирование:

```
Demo-Hero bot: Отлично! Укажи, пожалуйста, свой пол:

Demo-Hero bot: Отлично! Давай сначала познакомимся.
Скажи, чем ты занимаешься?

Пользователь: AI, nlp, агенты

Demo-Hero bot: Круто! Приятно познакомиться!
```

**Проблема:** нет вопросов о задачах гостя (инвестор? HR? партнёр?), нет понимания, что именно человек ищет на Demo Day.

---

## Tinder-механика

Бот показывает пары докладов и просит выбрать один:

```
Demo-Hero bot: Сейчас давай поиграем в игру -- я буду выводить тебе пары докладов
и супер-сжатую информацию о них, а ты будешь выбирать, какой именно из них тебе
больше нравится. И так до тех пор, пока тебе не надоест.

Начнём?

Demo-Hero bot: Подбираю пару докладов, немного подождите...

[карточки двух проектов]

Пользователь выбирает: "ИИ Таможенный Декларант"

Demo-Hero bot: Подбираю пару докладов, немного подождите...

Пользователь выбирает: "Цифровая бухгалтерия"

Demo-Hero bot: Подбираю пару докладов, немного подождите...

Пользователь выбирает: "Применение LLM для автоматизации бизнес-процессов ПАО Норильский Никель"
```

**Проблемы:**
- Медленно: нужно пролистать много пар, чтобы увидеть все ~330 проектов
- Нет контекста: непонятно, почему именно эти проекты в паре
- Нет фильтрации: нельзя сказать "покажи только FinTech" или "только агенты"
- Бинарный выбор: нельзя отложить оба или пропустить пару

---

## Расписание и маршрут

Накануне мероприятия:

```
Demo-Hero bot:
Привет, Дмитрий Горбунов! Завтра начинается конференция Demo Day!

Держи полное расписание конференции, чтобы ничего не упустить:

Пока в твоём маршруте нет докладов.

Я сделал твой индивидуальный маршрут конференции -- скажи,
если захочешь что-то изменить, и я это учту!
```

**Проблема:** маршрут пустой, хотя пользователь выбирал проекты в tinder-механике. Баг или потеря данных.

---

## Фидбэк

После доклада бот спрашивает оценку:

```
Demo-Hero bot: Расскажи, как тебе доклад "Дедупликация новостей"?
```

После мероприятия:

```
Demo-Hero bot:
Спасибо, что был с нами на Demo Day!

Мы надеемся, что ты ушёл с новыми знаниями и знакомствами.
Оцени, насколько тебе была полезна конференция (от 1 до 5).
```

**Проблема:** нет follow-up -- контактов, конспекта, next steps. Просто оценка 1-5.

## Выводы: что наш агент должен делать иначе

| Demo-Hero | Наш агент |
|-----------|-----------|
| Пол + "чем занимаешься" | Роль (инвестор/HR/партнёр) + конкретные задачи + отрасль/стек |
| Tinder-пары (медленно, ~330 проектов) | Мгновенная подборка по запросу с брифами |
| Пустой маршрут (баги) | Персональная программа с расписанием по залам |
| Нет Q&A помощи | Вопросы для авторов, матрица сравнения |
| Нет организации встреч | Слоты 1:1 и групповой Q&A |
| Оценка 1-5 после ивента | Follow-up пакет: контакты, конспект, next steps, шаблоны писем |

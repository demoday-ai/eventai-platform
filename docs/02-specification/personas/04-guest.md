## 4. Боли (Pain Points)

### 🔴 Острая боль (blocking problem):
> 330 проектов в 6-10 залах — невозможно сориентироваться. Видит максимум 16% проектов (34 из 212). Нет фильтрации по тематике, стеку, стадии. Выбирает наугад → попадает на нерелевантные проекты → теряет время → уходит разочарованная. Priority Score: 30/50.

### 🟡 Хроническая боль (ongoing frustration):
> Нет follow-up после мероприятия. Видела интересный проект, но забыла название. Нет контактов автора. Нет способа связаться после DD. Впечатления теряются через 2 дня.

### 🟢 Раздражающее неудобство (nice to fix):
> Demo-Hero не помог — Tinder-механика медленная, пары нерелевантные, маршрут с багами. "Фигня." Попытка использовать существующий инструмент привела к разочарованию.

## 5. Текущие решения

### Как решает проблему сейчас:
- **Наугад:** Приходит, смотрит расписание на стене, выбирает зал по названию тематики, заходит. Если не интересно — переходит в другой зал.
  - Плюсы: нет подготовки, zero friction
  - Минусы: случайность, пропускает 84% проектов, много нерелевантного
- **Demo-Hero (@jugru_conf_bot):** Бот с Tinder-механикой — показывает пары докладов, выбираешь. Формирует маршрут.
  - Плюсы: попытка решить проблему навигации
  - Минусы: "фигня", медленный, не масштабируется на 330 проектов, маршрут глючит
- **Контакты через организаторов:** Если понравился проект — просит организатора связать с автором.
  - Плюсы: персональное знакомство
  - Минусы: нужно найти организатора, он занят, контакт теряется

### Почему текущие решения не устраивают:
- "Максимально неудобное мероприятие для внешнего" (цитата организатора)
- Нет умной навигации — выбор наугад из 330 проектов
- Нет follow-up — контакты и впечатления теряются
- Низкая доходимость: из 190 регистраций реально ~50 приходят

## 8. Критерии успеха

### Как поймёт, что продукт работает:
- За 2 минуты в боте получила подборку 10-15 релевантных проектов с расписанием
- Каждый проект — краткий бриф (описание, стек, зал, время)
- После DD получила follow-up: контакты авторов, конспект, рекомендации

### Метрики успеха для этой персоны:
- Время профилирования: ≤2 минуты
- Релевантность подборки: >80% проектов по теме
- Конверсия: профилирование → посещение ≥3 проектов из подборки
- Follow-up получен в течение 24 часов после DD

## 9. Возражения и барьеры

### Типичные возражения:
- "Опять бот? Не хочу ничего настраивать" → Как преодолеть: 2 минуты свободным текстом в Telegram — не нужно ничего устанавливать, регистрироваться, заполнять формы
- "Подборка будет нерелевантная, как Demo-Hero" → Как преодолеть: AI-профилирование вместо Tinder-механики, учитывает контекст (стек, отрасль, задачу), показывает обоснование релевантности
- "Я и сама разберусь" → Как преодолеть: при 330 проектах в 6-10 параллельных залах — физически невозможно. Бот экономит 80% времени

### Барьеры к покупке:
- Низкая доходимость — может вообще не прийти (из 190 регистраций ~50 приходят)
- Скепсис после Demo-Hero — "боты для мероприятий не работают"
- Спонтанное решение прийти — может не знать о боте заранее
- Организаторы "уже на это не смотрят" — внешние гости не приоритет

### Аннотации

| # | Элемент | Поведение |
|---|---------|-----------|
| 1 | Дата + обратный отсчёт | Явная дата + «через N дней». ↗ иконка-ссылка → Импорт/Событие для редактирования |
| 2 | Quick Action (жёлтая секция) | Показывает ближайший незавершённый шаг. Кнопка «Перейти» ведёт на нужную страницу. Исчезает, когда всё завершено |
| 3 | Метрики (карточки) | 5 карточек в ряд. Числа обновляются auto-refresh |
| 4 | Таблица покрытия | Сводка по залам. ✓ = покрыт (зелёный), ⚠ = пробел (жёлтый). «Детали» → страница Эксперты/Покрытие конкретного зала |

### Состояния

- **Loading:** Скелетон вместо карточек и таблицы
- **Auto-refresh:** Каждые 30-60 сек, незаметно для пользователя, не сбрасывает скролл

### Переходы

- ↗ у даты → Импорт/Событие
- «Перейти» в Quick Action → зависит от текущего шага
- «Детали» → Эксперты/Покрытие (конкретный зал)

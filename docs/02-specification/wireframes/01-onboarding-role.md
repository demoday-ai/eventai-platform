# Wireframe: Онбординг — выбор роли

> **Платформа:** Mobile (Telegram bot, 375px)
> **Контекст:** Первый вход пользователя в бот
> **Цель пользователя:** Выбрать роль и начать сценарий

---

## Wireframe

```
╭────────────────────────────────────╮
│ Navigator                     ◯ ⚙ │
├────────────────────────────────────┤
│ Шаг 1 из 4                          │
│                                     │
│ Выберите свою роль:                 │
│                                     │
│ ( ) Организатор (по приглашению/    │
│     назначению)                     │
│ ( ) Эксперт / Жюри                  │
│ ( ) Участник проекта                │
│ ( ) Гость / Партнёр                 │
│                                     │
│ ╭────────────────────────────────╮  │
│ │ Код приглашения (если есть)    │  │
│ ╰────────────────────────────────╯  │
│ [ Запросить доступ у админа ]       │
│                                     │
│ ╭────────────────────────────────╮  │
│ │ Продолжить                     │  │
│ ╰────────────────────────────────╯  │
│                                     │
│ Подсказка: роль влияет на сценарии  │
╰────────────────────────────────────╯
```

---

## Аннотации

| # | Элемент | Поведение |
|---|---------|-----------|
| 1 | Выбор роли | Сохраняется в профиле пользователя |
| 2 | Код приглашения | Для организаторов по коду; опционально |
| 3 | Запросить доступ | Инструкция/ссылка для назначения админом |
| 4 | CTA «Продолжить» | Активируется после выбора роли (и валидного кода, если требуется) |

---

## Состояния

### Ошибка (роль не выбрана)

```
╭────────────────────────────────────╮
│ ⚠ Выберите роль, чтобы продолжить  │
╰────────────────────────────────────╯
```

### Ошибка (код приглашения неверный)

```
╭────────────────────────────────────╮
│ ⚠ Неверный код приглашения         │
╰────────────────────────────────────╯
```

### Ожидание назначения

```
╭────────────────────────────────────╮
│ ⏳ Запрос на доступ отправлен       │
│ Мы уведомим после назначения        │
╰────────────────────────────────────╯
```

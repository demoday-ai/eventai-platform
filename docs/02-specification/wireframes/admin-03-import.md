## Аннотации

| # | Элемент | Поведение |
|---|---------|-----------|
| 1 | Табы (1→2→3→4→5) | Все всегда доступны. Номера = рекомендуемый порядок. Активный таб подчёркнут |
| 2 | «Скачать шаблон XLSX» | Текстовая кнопка (secondary). Скачивает XLSX с заголовками + пример строки |
| 3 | Drag-area | Пунктирная рамка. Drag-and-drop или клик → file picker. После загрузки заменяется предпросмотром |
| 4 | Предпросмотр таблицы | Первые N строк + счётчик. Если ошибки — иконка ⚠ в строке |
| 5 | Inline-исправление привязки | Dropdown с существующими проектами. Подставляет наиболее похожий вариант |
| 6 | «Подтвердить» | Primary button. Disabled пока есть критические ошибки. Enabled когда все привязки корректны |
| 7 | «Загрузить другой» | Secondary button. Сбрасывает предпросмотр, показывает drag-area заново |

## Состояния

- **Empty state (таб Студенты, нет проектов):** «Для привязки студентов сначала загрузите проекты» + ссылка на таб «Проекты»
- **Empty state (нет события):** Все табы кроме «Событие» показывают: «Сначала создайте мероприятие» + ссылка на таб «Событие»

## Переходы

- Подтвердить (проекты) → автоматический переход на таб «Студенты»
- Подтвердить (студенты) → автоматический переход на таб «Эксперты»
- Подтвердить (последний таб) → Dashboard обновляет метрики

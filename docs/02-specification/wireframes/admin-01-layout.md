# Wireframe: Layout (Каркас)

**Контекст:** Общий каркас всех страниц админ-панели
**Цель пользователя:** Ориентироваться в системе, видеть прогресс, быстро переключаться между разделами

## Аннотации

| # | Элемент | Поведение |
|---|---------|-----------|
| 1 | Global Stepper (верхняя панель) | 3 фазы: ● заполненный = завершена, ○ пустой = не начата. Клик на фазу → переход к первой незавершённой странице этой фазы |
| 2 | ◯ @dima | Telegram-username текущего организатора. ▼ выпадающее меню: «Выйти» |
| 3 | Sidebar — группы | Визуально разделены тонким разделителем. Название группы — мелкий серый текст |
| 4 | Sidebar — статус-бейдж | ● зелёный = завершено, ○ серый = не начато, ⚠ жёлтый = требует внимания. Без бейджа = инструмент (Рассылки, Аудитория) |
| 5 | Sidebar — текущая страница | Фоновая подсветка строки + жирный текст |
| 6 | Content Area | Занимает всё пространство справа от sidebar. Скролл только внутри content area |

## Размеры

- Sidebar: фиксированная ширина ~220px
- Content Area: остальное пространство
- Top bar: фиксированная высота ~56px
- Минимальная ширина экрана: 1280px

# Wireframe: Эксперты

**Контекст:** Организатор распределяет экспертов по залам и контролирует покрытие
**Цель пользователя:** Обеспечить, чтобы каждый зал был покрыт достаточным количеством экспертов

## Аннотации

| # | Элемент | Поведение |
|---|---------|-----------|
| 1 | Табы (Матчинг, Список, Покрытие) | Все всегда доступны |
| 2 | Таблица матчинга | Залы + назначенные эксперты + статус (✓ / ⚠). Клик на зал → детали |
| 3 | Рекомендации | Появляются автоматически при наличии пробелов. Эксперты отсортированы по совпадению тегов |
| 4 | «Назначить» | Primary button. Мгновенно добавляет эксперта в зал, обновляет таблицу и статус |
| 5 | «Нет подходящих экспертов» | Если для зала нет кандидатов — текст: «Нет экспертов с подходящими компетенциями» |
| 6 | «Одобрить» | Активна только когда все пробелы закрыты ИЛИ организатор явно подтвердил пробелы |
| 7 | Детали зала (Покрытие) | Breadcrumb навигация. Список экспертов + таблица проектов |

## Состояния

- **Empty state (матчинг):** «Для матчинга одобрите кластеризацию» + ссылка → Кластеризация
- **Empty state (список):** «Загрузите экспертов на странице Импорта» + ссылка → Импорт/Эксперты

## Переходы

- Назначить эксперта → обновление таблицы + статуса зала
- Одобрить → подсказка «Следующий шаг — расписание» + ссылка
- Зал (клик) → детали зала на табе Покрытие
- ← Назад → возврат к сводке Покрытия

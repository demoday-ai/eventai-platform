# AI-агент-куратор DemoDay — Business Pivot

> Ветка для работы над бизнес-моделью и коммерциализацией

## Проблема

На Demo Day много проектов и треков (500+ участников, 300+ проектов), а времени у гостей мало: сложно быстро найти «свои» доклады/демо, сравнить проекты по нужным критериям и организовать содержательный Q&A.

## Фидбэк от организаторов кэмпа

> "мотивы понятны, но отсутствие цели коммерциализации мешает пройти весь путь на кэмпе (рекомендуется переформулировать цель/ценность и путь к результату)"

## Ключевые вопросы для проработки

1. **Кто платит?** Организаторы / акселераторы / HR / бизнес-партнёры?
2. **За что платят?** SaaS / white-label / доступ к талантам / premium follow-up?
3. **Сколько платят?** Цена за событие / подписка / за контакт?
4. **Как масштабировать?** Только AI Talent Camp или другие Demo Day?

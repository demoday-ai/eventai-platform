# Gemini CLI - AI-агент-куратор DemoDay

Этот файл содержит инструкции и контекст для Gemini CLI, оптимизированные для работы в репозитории `demoday-ai`.

## Обзор проекта
**AI-агент-куратор DemoDay (AI-first Unconference Navigator)** — платформа для автоматизации подготовки и проведения Demo Day. 
- **Цель:** Кластеризация проектов, подбор экспертов, персональная программа для гостей, Q&A-помощник и follow-up.
- **Команда:** "ЯСНОПОНЯТНО" (AI Talent Camp 2026, Проект #10).
- **Стек:** Python 3.12, FastAPI, PostgreSQL, SQLAlchemy 2.0, Alembic, python-telegram-bot, OpenRouter API.

## Основные директории
- `backend/`: Основной код (FastAPI + Telegram-бот).
- `docs/`: Обширная документация (Discovery, Specifications, C4, ERD).
- `data/`: Тестовые и сид-данные (анонимизированные Excel/JSON).
- `scripts/`: Скрипты для обработки данных.
- `specs/`: Планы реализации и задачи по фичам.

## Источники контекста
- `CLAUDE.md`: Детальный контекст по продукту, инфраструктуре и истории изменений.
- `README.md`: Краткий обзор и структура проекта.
- `AGENTS.md`: Правила работы и роли агентов в репозитории.
- `docs/02-specification/`: Техническое задание и архитектурные решения.

## Операционные правила для Gemini
1. **Язык:** Основной язык общения — русский. Код и комментарии (минимум) — на английском, следуя стилю проекта.
2. **Стиль кода:** Придерживайся существующих паттернов (FastAPI, SQLAlchemy 2.0 async, Pydantic v2).
3. **Безопасность:** Никогда не коммить секреты. Используй `.env` и `backend/app/config.py`.
4. **Тестирование:** Пиши тесты (pytest) для новых фич. Проверяй поведение, а не только реализацию.
5. **Документация:** При изменении функционала обновляй соответствующие файлы в `specs/` или `docs/`.
6. **Git:** Не делай `git commit` или `git push` без явного указания. Предлагай сообщение коммита.

## Команды и запуск (Backend)
- **Установка зависимостей:** `pip install -r backend/requirements.txt` (или через `pyproject.toml`).
- **Запуск миграций:** `alembic upgrade head` (в папке `backend`).
- **Запуск приложения:** `python -m app.main` или `uvicorn app.main:app --reload`.
- **Тесты:** `pytest`.
